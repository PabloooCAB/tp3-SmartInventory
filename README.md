# Smart Inventory API

Proyecto de gestión de inventario desarrollado con Spring Boot.

---

## 🚀 Tecnologías
- Java 21
- Spring Boot
- Maven
- Swagger / OpenAPI

---

## 📦 Funcionalidades
- CRUD de categorías
- CRUD de productos
- Movimientos de stock (entrada/salida)
- Alertas de stock bajo
- Reporte de performance

---

## ▶️ Cómo ejecutar

```bash
mvn spring-boot:run
